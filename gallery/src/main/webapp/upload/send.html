서버로 파일을 전송하는 업로드라는 기술은 java로 직접 구현할 경우, 개발자가 네트워크 통신, IO(입출력), File 등을
		 제어해야 한다.. 생산성이 떨어지므로, 실무에서는 이미 누군가가 제작해놓은 라이브러리를 이용함
		 
		 [ 종류 ]
		 유료 - ..
		 무료 - Apache에서 제작한 FileUpload 컴포넌트,
		 		Oreilly 라는 IT 출판사에서 Servlet 책을 발간하면서 예제로 제작한 업로드 컴포넌트인 Cos.jar가 있음
		 		https://www.servlets.com/cos/
		 		Cos.jar 복사해서 붙여넣기 WEB-INF-lib
서버로 일반 텍스트가 아닌, 이미지 동영상과 같은 파일(인간이 해석할 수 없는 유형의 파일을 바이너리 파일)은 반드시 POST 방식으로 전송해야 하며, 특히나 인코딩 타입도 multipart/form-data 라는 형식을 지정해야 한다.. 이때 multipart의 의미는 서버로 전송하고자 하는 미디어 타입이 텍스트뿐만 아니라, 바이너리도 섞여 있는 복합유형을 의미
